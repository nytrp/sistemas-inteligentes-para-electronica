📌 CHULETA BÁSICA DE GIT

1) Traer lo último del repositorio:
   git pull origin main

2) Ver estado de cambios:
   git status

3) Agregar todos los cambios:
   git add .

4) Guardar cambios localmente:
   git commit -m "mensaje claro"

5) Subir cambios a GitHub:
   git push origin main

6) Crear nueva rama (opcional):
   git checkout -b nombre-rama

7) Cambiar de rama:
   git checkout main

🔥 Recuerda: Siempre haz 'git pull' antes de trabajar para evitar conflictos.
